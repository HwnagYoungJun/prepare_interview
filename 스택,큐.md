# Stack, Queue



## Stack -> Queue

### 코드(Python)

```python
class Stack:
    
    def __init__(self):
        self.items = []
        
    def is_empty(self):
        return self.items == []
    
    def push(self, item):
        self.items.append(item)
       
    def pop(self):
        if not self.items:
            return -1
        else:
            return self.items.pop()
        
class MyQueue:
    
    def __init__(self):
        self.stack1 = Stack()
		self.stack2 = Stack()
        
    def enqeue(self, item):
        self.stack1.push(item)
        
    def dequeue(self):
        if self.stack2.is_empty():
        	while self.stack1:
            	self.stack2.push(self.stack1.pop())
        
        return self.stack2.pop()
```
